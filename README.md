# cross-table

## Setup
* **Node**
```
npm init -y
```

* **TypeScript**
```
npm install -g typescript
```

* **TypeScript ESLint**
```
npm install --save-dev @typescript-eslint/parser @typescript-eslint/eslint-plugin eslint typescript
```

* **TailWindCSS**
```
npm install --save-dev tailwindcss
```

To create a JS file from TS `tsc app.ts`
